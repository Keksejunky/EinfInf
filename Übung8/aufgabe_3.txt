Rekurrenzgleichungen für i)
    t1 = 1
    t2 = c + 2*t1
    tn = c + t1 + t(n-2)            falls n >= 3

Rekurrenzgleichungen für ii)
    t1 = 1
    t2 = c + 2*t1
    tn = c + 2*t(n/2)               falls n >= 3

Rekurrenzgleichungen für iii)
    t1 = 1
    t2 = c + t1
    tn = c + 2tn - 1

Rekurrenzgleichungen für iv)
    t1 = 1
    t2 = 1
    tn = c + 2*t(n-1)

b) 

Geschlossene Form + Begründung für i)
    tn = n
    nur der gewinner kommt weiter, also muss man die verlierer nicht weiter testen. Also müssen nur so viele vergleiche durchgeführt werden, wie es spieler gibt.

Geschlossene Form + Begründung für ii)
    tn = log2(n)
    in jeder Runde gibt es nur halb so viele teilnehmer wie in der Runde davor. Also müssen immer nur halb so viele vergleiche durchgeführt werden.

Geschlossene Form + Begründung für iii)
    tn = 2n - 1
    jeder muss gegen a und gegen b spielen, da es egal ist ob a gegeb b oder b gegen a spielt müssen nicht ganz 2n durchläufe gemacht werden.

Geschlossene Form + Begründung für iv)
    tn = n^2
